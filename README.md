# Quri 

Quri is a full-stack web application with a clean, Google-like search interface.
It fetches real-time data from Wikipedia using a public API and is deployed live.

##Live Demo
https://quri.onrender.com

##Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- API: Wikipedia Search API
- Deployment: Render

##Features
- Real-time search results
- Keyboard (Enter key) support
- Clean minimal UI
- Fully deployed web application

##Deployment
The application is deployed on Render and automatically redeploys on every GitHub push.
